☕ BeanLog - 나만의 커피 아카이브 & 브루잉 타이머

서비스 접속: https://imonraside.github.io/beanlog

BeanLog는 커피 애호가를 위한 올인원 기록 앱입니다. 원두 구매부터 시음 노트 작성, 그리고 정교한 브루잉 타이머까지 하나의 앱에서 관리하세요. PWA(Progressive Web App) 기술로 제작되어 별도의 스토어 다운로드 없이 웹에서 바로 설치하여 네이티브 앱처럼 사용할 수 있습니다.

📲 앱 설치 방법

BeanLog는 PWA 앱입니다. 브라우저를 통해 서비스 주소에 접속한 뒤 홈 화면에 추가하여 사용하세요.

Android (삼성 인터넷 / Chrome)

  1. 브라우저 주소창 옆의 설치 아이콘(⬇️) 또는 하단의 '앱 다운로드' 배너를 클릭하세요.
  2. 또는 메뉴(⋮) > '앱 설치' 혹은 '홈 화면에 추가'를 선택하세요.

iOS (Safari)

  1. Safari 브라우저 하단의 공유 버튼(네모 위 화살표)을 누르세요.
  2. 메뉴에서 '홈 화면에 추가'를 선택하세요.

🌟 주요 기능 소개

1. 원두 아카이브 (Beans)

내가 마신 원두의 정보를 체계적으로 기록하고 관리합니다.

  • AI 자동 입력 (OCR): 원두 패키지 사진을 찍으면 AI(Gemini)가 원두 이름, 지역, 가공 방식, 컵 노트 등을 자동으로 분석하여 입력해 줍니다. (설정에서 API 키 필요)
  • 상세 정보 기록: 나라이름, 지역, 품종, 가공 방식, 고도, 로스팅 포인트, 구매처, 가격 등을 꼼꼼하게 기록할 수 있습니다.
  • 블렌드 정보: 블렌딩된 원두의 구성 비율과 품종 정보를 상세하게 기록할 수 있습니다.
  • 상태 및 정렬: 원두 소진 여부(Sold Out) 관리와 다양한 정렬(구매일, 평점, 로스팅순 등) 기능을 제공합니다.
  • 랜덤 픽 (Today's Pick): 무엇을 마실지 고민될 때, 보유 중인 원두 중 하나를 추천해 줍니다. (메인 화면 ✨ 아이콘)
  • 원두 공유: 원두 정보를 이미지 카드, 텍스트, 또는 링크로 공유할 수 있습니다. 특히 링크 공유 기능을 사용하면, 친구가 링크를 클릭하는 것만으로 내 원두 정보를 자신의 앱에 바로 저장할 수 있습니다.

2. 시음 노트 (Tasting Note)

커피의 맛을 감각적으로 기록하고 공유하세요.

  • 레이더 차트: 산미, 밸런스, 단맛, 클린컵, 바디, 향미 6가지 항목을 점수화하여 시각적인 오각형 차트로 보여줍니다.
  • 플레이버 피커 (Flavor Picker): 과일, 꽃, 견과류 등 8가지 카테고리, 80여 개의 향미 태그를 터치만으로 간편하게 선택할 수 있습니다.
  • 시음 기록 공유: 내가 기록한 맛 점수(레이더 차트)와 노트가 담긴 이미지 카드를 생성하여 인스타그램 등에 공유하거나, 상세 내용을 텍스트로 복사할 수 있습니다.

3. 브루잉 타이머 (Brewing Timer)

복잡한 레시피도 실수 없이 내릴 수 있도록 도와줍니다.

  • 단계별 가이드: 물 붓는 양, 시간, 행동(젓기, 스위치 조작 등)을 단계별로 큰 화면으로 안내합니다.
  • 커스텀 레시피: 나만의 레시피를 특수 문법을 사용해 자유롭게 만들고 저장할 수 있습니다.
  • 음성/화면 유지: 타이머 작동 중에는 화면이 꺼지지 않습니다. (Wake Lock 지원)

4. 인사이트 (Insights)

나의 커피 생활을 통계로 확인하세요.

  • 대시보드: 누적 구매량, 소비량, 총 시음 횟수, 보유 원두 현황 등을 한눈에 볼 수 있습니다.
  • 지출 분석: 원두 구매 비용과 카페 음료 지출을 구분하여 상세한 금액 통계를 제공합니다.
  • 커피 캘린더: 언제 커피를 마셨는지 히트맵(Heatmap) 달력으로 보여줍니다.
  • 기록 탐색: 달력의 날짜를 누르면 그날 마신 커피 기록으로 바로 이동합니다.

⏱️ 타이머 레시피 문법 가이드

BeanLog 타이머는 텍스트만 입력하면 자동으로 단계를 생성하는 강력한 파싱 기능을 제공합니다.

기본 구조

  • 원두: 20 (숫자만 입력 시 g 단위 자동 인식)
  • 물: 60 80 80 80 (띄어쓰기로 구분하면 단계별 물 투입량이 됩니다)
  • 시간: 30 40 40 (각 단계별 소요 시간, 초 단위)
  • 종료 시간: 3:00 (종료 시간, 분 단위 or 초 단위)

고급 명령어 (특수 동작)

물 투입량 옆에 아래 코드를 붙여서 특수 동작을 지정할 수 있습니다.

코드 동작 아이콘 설명 예시
cl Close 🔒 클레버/스위치 닫기 cl60 (닫고 60ml 붓기)
op Open 🔓 클레버/스위치 열기 op (추출구 열기)
st Stir 🥄 교반 (젓기) st (스푼으로 젓기)
sw Swirl 👋 스월링 (흔들기) sw (드리퍼 흔들기)
c Circle ⭕ 원형 푸어 c60 (원 그리며 60ml)
s Spiral 🌀 나선형 푸어 s60 (나선형으로 60ml)
ct Center 🎯 중앙 푸어 ct100 (가운데만 100ml)

📝 레시피 작성 예시

[하리오 스위치 레시피]

  • 물 입력: cl60 60 op 100
  • 해석:
    1. 스위치를 닫고(cl) 물 60ml 투입
    2. 물 60ml 추가 투입
    3. 스위치를 열고(op)
    4. 물 100ml 추가 투입

⚙️ 설정 및 데이터 관리

Google Gemini API 설정 (AI OCR)

원두 사진 자동 분석 기능을 사용하려면 Google Gemini API 키가 필요합니다.

  1. Google AI Studio에서 무료 API 키를 발급받으세요.
  2. 앱 내 설정(Settings) 탭 > Gemini API Key 항목에 입력하세요.
  3. 키는 사용자의 브라우저(로컬 스토리지)에만 저장되며 외부로 전송되지 않습니다.

데이터 백업 및 복원

BeanLog는 서버 없이 동작하는 앱으로, 모든 데이터는 사용자의 브라우저 저장소(LocalStorage)에 저장됩니다. 브라우저 캐시나 데이터를 삭제하면 저장된 내용이 함께 사라질 수 있으므로, 중요한 데이터는 주기적으로 백업해 두는 것을 권장합니다.

  • 백업: 설정 탭에서 '통합 백업'을 누르면 .json 파일로 데이터가 다운로드됩니다.
  • 복원: '복원' 버튼을 눌러 백업 파일을 선택하면 데이터를 불러옵니다. (기기 변경 시 유용)
  • 데모 데이터: 앱의 기능을 미리 체험해볼 수 있도록 샘플 데이터를 제공합니다.

🚀 업데이트 내역 (Changelog)

v1.1.5

  • 시스템 개선: 데이터 저장소를 로컬 스토리지에서 IndexedDB로 변경하여 성능과 안정성을 높였습니다. (기존 데이터 자동 마이그레이션)
  • 통계 탭 수정: 원두 구매 정보와 한잔 커피 가격이 함께 있는 경우, 통계 집계에서 누락되던 문제를 해결했습니다.
  • 정렬 로직 개선: '최근 로스팅순' 정렬 시 무게 정보가 있는 원두를 상단에, 한잔 커피 기록을 하단에 배치하도록 개선했습니다.
  • UI 버그 수정: 통계 탭의 상세 팝업이 열린 상태에서 다른 탭으로 이동했다 돌아오면 팝업이 닫히도록 수정했습니다.

v1.1.4

  • 입력 편의성 개선: 노트 작성 시 내용에 따라 텍스트 창 높이가 자동으로 조절되며, 구매처 입력 시 추천 목록 UI가 개선되었습니다.
  • URL 자동 입력: 기존에 입력했던 구매처를 선택하면 저장된 쇼핑몰 URL을 자동으로 불러옵니다.
  • 이미지 편집 수정: 모바일 환경에서 원두 사진 크롭 시 이미지가 잘리거나 영역을 벗어나는 문제를 해결했습니다.
  • 통계 로직 수정: 캘린더의 '구매' 횟수 집계 시 카페 음료를 제외하고 원두(Weight)만 카운트하도록 변경되었습니다.
  • 시스템 업데이트: AI OCR 모델을 최신 버전(Gemini 2.5 Flash)으로 업그레이드하여 안정성을 높였습니다.

v1.1.3

  • 블렌드 정보 추가: 원두 등록 시 블렌딩 비율과 품종 정보를 상세하게 기록할 수 있습니다.
  • 지출 분석 기능: 통계 탭에서 원두 구매 비용과 카페 지출을 구분하여 볼 수 있습니다.
  • 공유 기능 강화: 원두 정보 및 시음 기록을 이미지 카드, 텍스트, 앱 링크로 공유할 수 있습니다.
  • 데모 데이터: 앱 기능을 미리 체험해볼 수 있는 샘플 데이터 추가 기능을 제공합니다.
  • UI 개선: 다크 모드 지원 및 전반적인 디자인이 개선되었습니다.

💡 Tips

  • 앱 아이콘: 홈 화면에 추가된 아이콘을 길게 누르면 앱 정보 등을 확인할 수 있습니다.
  • 간편 메뉴: 원두 목록이나 시음 기록을 길게 누르면(Long Press) 공유 및 삭제 메뉴가 나타납니다.
  • 검색: 원두 목록 상단의 검색창을 통해 원두 이름, 컵 노트, 지역 등을 검색할 수 있습니다.
  • 다크 모드: 시스템 설정을 따르거나, 설정 탭에서 라이트/다크 모드를 직접 선택할 수 있습니다.
