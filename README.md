# ☕ BeanLog - 나만의 커피 아카이브 & 브루잉 타이머

**BeanLog**는 커피 애호가를 위한 올인원 기록 앱입니다. 원두 구매부터 시음 노트 작성, 그리고 정교한 브루잉 타이머까지 하나의 앱에서 관리하세요. PWA(Progressive Web App) 기술로 제작되어 별도의 스토어 다운로드 없이 웹에서 바로 설치하여 네이티브 앱처럼 사용할 수 있습니다.

## 📲 앱 설치 방법

BeanLog는 **PWA** 앱입니다. 브라우저를 통해 접속한 뒤 홈 화면에 추가하여 사용하세요.

### Android (삼성 인터넷 / Chrome)
1. 브라우저 주소창 옆의 **설치 아이콘(⬇️)** 또는 하단의 **'앱 다운로드'** 배너를 클릭하세요.
2. 또는 메뉴(⋮) > **'앱 설치'** 혹은 **'홈 화면에 추가'**를 선택하세요.

### iOS (Safari)
1. Safari 브라우저 하단의 **공유 버튼(네모 위 화살표)**을 누르세요.
2. 메뉴에서 **'홈 화면에 추가'**를 선택하세요.

## 🌟 주요 기능 소개

### 1. 원두 아카이브 (Beans)
내가 마신 원두의 정보를 체계적으로 기록하고 관리합니다.

*   **AI 자동 입력 (OCR):** 원두 패키지 사진을 찍으면 AI(Gemini)가 원두 이름, 지역, 가공 방식, 컵 노트 등을 자동으로 분석하여 입력해 줍니다. (설정에서 API 키 필요)
*   **상세 정보 기록:** 나라이름, 지역, 품종, 가공 방식, 고도, 로스팅 포인트, 구매처, 가격 등을 꼼꼼하게 기록할 수 있습니다.
*   **상태 관리:** 원두가 남았는지, 다 마셨는지(Sold Out) 상태를 관리하여 보유 중인 원두만 필터링해 볼 수 있습니다.
*   **랜덤 픽 (Today's Pick):** 무엇을 마실지 고민될 때, 보유 중인 원두 중 하나를 추천해 줍니다. (메인 화면 ✨ 아이콘)

### 2. 시음 노트 (Tasting Note)
커피의 맛을 감각적으로 기록하고 공유하세요.

*   **레이더 차트:** 산미, 밸런스, 단맛, 클린컵, 바디, 향미 6가지 항목을 점수화하여 시각적인 오각형 차트로 보여줍니다.
*   **플레이버 피커 (Flavor Picker):** 과일, 꽃, 견과류 등 8가지 카테고리, 80여 개의 향미 태그를 터치만으로 간편하게 선택할 수 있습니다.
*   **공유 카드 생성:** 내가 기록한 원두 정보와 시음 평점이 담긴 예쁜 이미지를 생성하여 인스타그램이나 커뮤니티에 공유할 수 있습니다.

### 3. 브루잉 타이머 (Brewing Timer)
복잡한 레시피도 실수 없이 내릴 수 있도록 도와줍니다.

*   **단계별 가이드:** 물 붓는 양, 시간, 행동(젓기, 스위치 조작 등)을 단계별로 큰 화면으로 안내합니다.
*   **커스텀 레시피:** 나만의 레시피를 특수 문법을 사용해 자유롭게 만들고 저장할 수 있습니다.
*   **음성/화면 유지:** 타이머 작동 중에는 화면이 꺼지지 않습니다. (Wake Lock 지원)

### 4. 인사이트 (Insights)
나의 커피 생활을 통계로 확인하세요.

*   **대시보드:** 누적 구매량, 소비량, 총 시음 횟수 등을 한눈에 볼 수 있습니다.
*   **커피 캘린더:** 언제 커피를 마셨는지 히트맵(Heatmap) 달력으로 보여줍니다.
*   **기록 탐색:** 달력의 날짜를 누르면 그날 마신 커피 기록으로 바로 이동합니다.

## ⏱️ 타이머 레시피 문법 가이드

BeanLog 타이머는 텍스트만 입력하면 자동으로 단계를 생성하는 강력한 파싱 기능을 제공합니다.

### 기본 구조
*   **원두:** `20` (숫자만 입력 시 g 단위 자동 인식)
*   **물:** `60 80 80` (띄어쓰기로 구분하면 단계별 물 투입량이 됩니다)
*   **시간:** `30 40 40` (각 단계별 소요 시간, 초 단위)

### 고급 명령어 (특수 동작)
물 투입량 옆에 아래 코드를 붙여서 특수 동작을 지정할 수 있습니다.

| 코드 | 동작 | 아이콘 | 설명 | 예시 |
|:---:|:---:|:---:|:---|:---|
| **cl** | Close | 🔒 | 클레버/스위치 닫기 | `cl60` (닫고 60ml 붓기) |
| **op** | Open | 🔓 | 클레버/스위치 열기 | `op` (추출구 열기) |
| **st** | Stir | 🥄 | 교반 (젓기) | `st` (스푼으로 젓기) |
| **sw** | Swirl | 👋 | 스월링 (흔들기) | `sw` (드리퍼 흔들기) |
| **c** | Circle | ⭕ | 원형 푸어 | `c60` (원 그리며 60ml) |
| **s** | Spiral | 🌀 | 나선형 푸어 | `s60` (나선형으로 60ml) |
| **ct** | Center | 🎯 | 중앙 푸어 | `ct100` (가운데만 100ml) |

#### 📝 레시피 작성 예시
**[하리오 스위치 레시피]**
*   **물 입력:** `cl60 60 op 100`
*   **해석:**
    1. 스위치를 **닫고(cl)** 물 **60ml** 투입
    2. 물 **60ml** 추가 투입
    3. 스위치를 **열고(op)**
    4. 물 **100ml** 추가 투입

## ⚙️ 설정 및 데이터 관리

### Google Gemini API 설정 (AI OCR)
원두 사진 자동 분석 기능을 사용하려면 Google Gemini API 키가 필요합니다.
1. Google AI Studio에서 무료 API 키를 발급받으세요.
2. 앱 내 **설정(Settings)** 탭 > **Gemini API Key** 항목에 입력하세요.
3. 키는 사용자의 브라우저(로컬 스토리지)에만 저장되며 외부로 전송되지 않습니다.

### 데이터 백업 및 복원
BeanLog는 서버 없이 동작하는 앱으로, 모든 데이터는 사용자의 기기에 저장됩니다.
*   **백업:** 설정 탭에서 **'통합 백업'**을 누르면 `.json` 파일로 데이터가 다운로드됩니다.
*   **복원:** **'복원'** 버튼을 눌러 백업 파일을 선택하면 데이터를 불러옵니다. (기기 변경 시 유용)
